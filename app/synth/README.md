# Dynamic XY Web Synth & Sequencer
https://nyseer-portfolio.vercel.app/synth

* High-performance analog-modeled synthesizer using the Web Audio API and Tone.js, featuring custom oscillators, ADSR envelopes, and resonant filters.
* Custom dynamic XY touchpad component to smoothly slide through notes and various effects. 
* Optimized UI/UX for low-latency interaction, and efficient use of DOM rendering practices in React to maintain low latency while processing complex audio graphs.
* Significantly reduced initial webpage load time by code-splitting pages and lazy loading synth components. 
* State management to handle real-time parameter modulation without audio artifacts.
* Precise custom 16-step sequencer with global clock sync, ensuring low-latency playback and sample-accurate timing across multiple browser environments.
